# Modules médecins

Ce dossier contient tous les modules de télémédecine accessibles au médecin (consultation à distance, téléconsultation, téléexpertise, téléassistance, télésoin, télémonitoring, e-prescription, gestion RDV/facturation, IA avancée, blockchain, etc.).

Chaque fichier HTML correspond à un module ou sous-module du parcours médecin.
