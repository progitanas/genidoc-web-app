# Modules établissements

Ce dossier contient tous les modules de télémédecine accessibles à l'établissement (consultation à distance, téléconsultation, téléexpertise, téléassistance, télésoin, télémonitoring, e-prescription, gestion RDV/facturation, IA avancée, blockchain, etc.).

Chaque fichier HTML correspond à un module ou sous-module du parcours établissement.
